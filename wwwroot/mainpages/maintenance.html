Programs used:

The HTML code was generated using Dreamweaver 4. Three templates were used to create the three types of pages. The buttons were created using Fireworks 4, and images were edited there too. The movies were created using Flash 5 and Swish. The Site Maps were created in Mind Mananger, and then altered in Fireworks and saved as png files.
